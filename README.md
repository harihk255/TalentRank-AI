# 🚀 TalentRank-AI
### AI-Powered Resume Ranking & Candidate Screening System

TalentRank-AI is an intelligent recruitment solution that automates the resume screening process using Artificial Intelligence and Machine Learning. The system analyzes candidate profiles, compares them with job requirements, ranks applicants based on relevance, and presents actionable insights through an interactive Power BI dashboard.

The project is designed to reduce manual effort for recruiters, improve hiring efficiency, and enable data-driven recruitment decisions.

---

## 📌 Project Overview

Recruiters often spend hours manually reviewing hundreds of resumes for a single job opening. TalentRank-AI streamlines this process by:

- Parsing candidate data
- Comparing resumes against job requirements
- Ranking candidates automatically
- Generating recruiter-friendly insights
- Visualizing hiring metrics in Power BI

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/harihk255/TalentRank-AI.git
```

Move into the project

```bash
cd TalentRank-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python rank.py
```

Generate dashboard data

```bash
python generate_dashboard_data.py
```

Open the Power BI dashboard to explore the generated analytics.
