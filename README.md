# TalentRank-AI

An AI-powered Resume Ranking and Candidate Screening System that helps recruiters automatically rank resumes based on job descriptions using Natural Language Processing (NLP) and Machine Learning.

## Features

- Resume ranking based on job description
- AI-powered candidate scoring
- Resume parsing and preprocessing
- Skill matching
- Candidate dashboard
- Interactive frontend
- Fast and scalable ranking system

## Tech Stack

### Frontend

## Installation

Clone the repository

```bash
git clone https://github.com/harihk255/TalentRank-AI.git
```

Move into the project

```bash
cd TalentRank-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python rank.py
```

## Workflow

1. Upload resumes
2. Upload Job Description
3. AI processes all resumes
4. Candidates are scored
5. Dashboard displays rankings
6. Recruiters select the best candidates
